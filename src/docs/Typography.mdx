# Typography & Fonts

Three faces carry the system. They are loaded by the `theme` item (Google-hosted woff2,
latin subset) and mapped to Tailwind v4 font utilities (`font-display`, `font-sans`,
`font-mono`).

## Type scale

Rendered with the registry `Typography` component (`@crashoverride/typography`). Headings
ride the display face; body is Inter; inline code is Geist Mono.

## Usage

```tsx
import { H1, P, InlineCode } from "@/registry/crashoverride/ui/typography"

<H1>Tag. Track. Trust.</H1>
<P>Body copy in Inter. <InlineCode>chalk insert</InlineCode>.</P>
```

> **Casing:** Sentence case for body and most headings. UPPERCASE with wide letter-spacing
> is reserved for eyebrow labels and status pills (`PLATFORM CAPABILITIES`, `01 · MONITOR`).
> Never set a full sentence in all-caps.
