# Spacing & Radius

## Spacing scale

An **8px grid** with half-steps for fine control. Use it for padding, gaps, and layout
rhythm — Tailwind's spacing utilities (`p-4`, `gap-6`, `mt-2`) map onto these values.

## Radius

The brand radius is a **sharp `0.25rem` (4px)** — corners are crisp, never pill-soft.
Components read `--radius`; Tailwind's `rounded-sm / -md / -lg` are derived from it
(`calc(--radius − 2px)`, `--radius`, `calc(--radius + 2px)`).

<div className="flex gap-8">
  <RadiusBox label="rounded-sm" radius="calc(0.25rem - 2px)" />
  <RadiusBox label="rounded-md (--radius)" radius="0.25rem" />
  <RadiusBox label="rounded-lg" radius="calc(0.25rem + 2px)" />
</div>

> **Surfaces, not shadows.** Cards and panels separate from the page with a **hairline
> border** and the sharp radius — never a heavy drop shadow. The page is brand Black
> `#161616`; raised cards sit on a slate surface and pop via the 1px border.
