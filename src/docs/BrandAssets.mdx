# Brand Assets

The official Crash Override logo art, shipped with the design system. Each mark is rendered
at a real size on an appropriate backdrop — white-fill marks on the brand black, black-fill
marks on Thunder white — so the contrast you would actually ship is what you see here.

The local `public/brand-assets/` copy was deleted; the canon now lives in
`@crashoverride/brand-tokens` (the brand-visual repo). Pull the exact file you need;
do not recolour or redraw the art.

## Icon mark

The "needle" mark on its own. Use it for favicons, app icons, avatars, and tight spaces
where the wordmark would not be legible.

## Code logo (`<Logo />`)

The in-repo React component (`@/registry/crashoverride/ui/logo`). The mark is inline SVG
drawn in `currentColor`, so it tints to the surrounding text colour — lime on the dark
theme, fandango on light. This is the **code version**: prefer it in the app so the mark
themes automatically; use the static files above for favicons, social images, and print.
