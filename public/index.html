Crash Override · shadcn registry

React Design System

The Crash Override Design System packaged as a public shadcn registry — a brand theme plus 60+ branded components and composite blocks, installable into any shadcn app.

Tag. Track. Trust.

Consume the registry

Point your components.json at the @crashoverride namespace:

// components.json
{
  "registries": {
    "@crashoverride": "https://crashappsec.github.io/react-registry/r/{name}.json"
  }
}

Install the theme first (it carries the brand tokens + fonts), then add components and blocks:

# the theme is the keystone — install it before anything else
npx shadcn add @crashoverride/theme

# then any component or block
npx shadcn add @crashoverride/button
npx shadcn add @crashoverride/stat-card-row

Brand in one breath

Dark-first surfaces · Atomic-lime primary (Fandango in light) · JetBrains Mono (display) / Inter (body) / Geist Mono (mono) · sharp 4px radius. The theme item ships the oklch light+dark bridge so every component tracks both modes automatically.
